## Usage

1. Go to [releases](https://github.com/catppuccin/vivaldi/releases).
2. Download your favourite variant and colour tarball.
3. Launch Vivaldi, then go to `Settings` -> `Themes` -> `Open Theme…` and open a downloaded tarball.
